# iDragon - JavaScript Game

iDragon is an exciting JavaScript adventure game inspired by classic endless runner games. Take control of the Dino, dodge obstacles (dragons), and see how high you can score!

## Features

- Responsive gameplay with jump and move controls
- Dynamic obstacles with increasing difficulty
- Score tracking system
- Game over detection with restart option
- Background music and sound effects

## How to Play

- Press the **Up Arrow (↑)** to jump.
- Press the **Right Arrow (→)** to move right.
- Press the **Left Arrow (←)** to move left.
- Avoid colliding with incoming dragons (obstacles).
- Try to achieve the highest score possible!

## Setup

1. Clone or download this repository.
2. Ensure you have the following file structure:
   - `index.html`
   - `style.css`
   - `app.js`
   - `img/` directory with `bg.png`, `dino.png`, and `dragon.png`
   - `music/` directory with `music.mp3` and `gameover.mp3`
3. Open `index.html` in your web browser to start playing!
